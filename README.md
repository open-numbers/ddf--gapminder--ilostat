# ddf--gapminder--ilostat

subset of ILOSTAT dataset in DDF format

These indicators are part of [Systema Globalis](https://github.com/open-numbers/ddf--gapminder--systema_globalis).

# How to update

To update this dataset, first update below dataset(s) and run the etl recipe in this dataset.

- [open-numbers/ddf--ilo--ilostat](https://github.com/open-numbers/ddf--ilo--ilostat)
